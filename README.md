# Slides pour atelier Paris Web 2019 "N'ayez plus peur du terminal !"

Getting started
---

```shell script
$ npm install reveal-md
```

Afficher les slides
---

```shell script
$ make start
```

Générer les slides
---

```
$ make build-static
``` 


Enrigistrer des asciinema
---

```shell script
asciinema rec data/title.cast --overwrite
```

Afficher les touches pressées
---
